# 数据预处理与质量清洗示例

编号 01–08 是界面“加载示例数据”实际使用的测试集：

- 01 钻孔：重复行、缺失岩性/水位、深度异常；
- 02 剖面：字段缺失和重复记录；
- 03 点云：重复点、分类缺失和高程异常；
- 04 栅格像元表：缺失像元、重复像元和异常值；
- 05 GeoJSON：属性模式不一致、重复要素；
- 06 OBJ：重复顶点、非法面和开放边；
- 07 正常对照：用于验证清洗不会无故修改正常数据；
- 08 单位归一：`mm→m`、`MPa→Pa`、`deg→rad`，并验证字段补全与去重。

自动清洗始终写入 `output/preprocessing`，不会修改源文件。每次运行生成：

1. 各输入文件对应的 `_cleaned` 成果；
2. 清洗动作明细，包括去重、缺失填补、异常截断、单位归一和网格修复；
3. 输入/输出 SHA-256；
4. `cleaning_manifest.json`；
5. 清洗后自动复检评分。

开放网格不会被程序凭空补面，因为自动猜测缺失几何可能改变工程含义；该问题会保留为“需人工复核”。
